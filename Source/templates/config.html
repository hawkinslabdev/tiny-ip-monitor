  - WEBHOOK_METHOD={{ config.WEBHOOK_METHOD }}
  - WEBHOOK_USER={{ config.WEBHOOK_USER if config.WEBHOOK_USER else '(not set)' }}
  - WEBHOOK_PASS={{ '***' if config.WEBHOOK_PASS else '(not set)' }}
  - CRON_SCHEDULE=0 */12 * * *
  - TZ=UTC
  - WEB_PORT=8080

How to Update Configuration

To modify these settings:

  1. Edit the environment variables in your docker-compose.yml file
  2. Stop the container: docker-compose down
  3. Start with new configuration: docker-compose up -d
  4. Monitor logs: docker-compose logs -f
